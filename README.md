## You Are Empty script dump
These scripts were authored by Vyacheslav Korotayev & Yuri Dobronravin.

### Preprocessor
It seems like they had an in-house preprocessor. Neat. If I find any more information on it I will include it here.

### Why?
Going through the internals of games from a time before off-the-shelf engines were the norm is a hobby of mine. It's interesting to me to see the internals. I like to see the decisions made, the stylistic choices, and above all I appreciate the craftsmanship that used to be commonplace in software. I hope you can too.
